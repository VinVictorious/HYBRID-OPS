Install HYBRID OPS

Add the app to your Home screen for a fast, full-screen experience.

If your browser offers an install prompt, use it here:

If the button is disabled or hidden, follow the steps below for your device.

Android (Chrome)

  1. Open HYBRID OPS in Chrome.
  2. Tap the menu (⋮) in the top-right corner.
  3. Select Add to Home screen or Install app.
  4. Confirm by tapping Install.

iOS (Safari)

  1. Open HYBRID OPS in Safari.
  2. Tap the Share button (square with an up arrow).
  3. Choose Add to Home Screen.
  4. Tap Add to confirm.

Desktop (Chrome/Edge)

  1. Open HYBRID OPS in Chrome or Edge.
  2. Click the Install icon in the address bar, or open the browser menu.
  3. Select Install or Install app, then confirm.
